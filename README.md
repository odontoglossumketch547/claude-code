## 🧰 Troubleshooting

If the app does not open:

- Check that the download finished
- Make sure you extracted the ZIP file if you downloaded one
- Run the file again as a normal user first
- Try right-clicking the app and choosing **Run as administrator**
- Check that Windows Defender did not block the file

If the app opens but does not connect:

- Check your internet connection
- Check your API key or account details
- Try again after closing and reopening the app
- Make sure you picked the right project folder

If the app shows missing Python files:

- Install Python 3.11 or newer
- Make sure Python is in your PATH
- Reopen the app after the install
